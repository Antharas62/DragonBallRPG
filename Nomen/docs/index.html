N.O.M.E.N - A new Opensource Mugen Editor made by Nobun

Nomen is an editor for creating / editing Mugen works (chars, stages, etc). One of the most relevant difference from other Mugen editors is that Nomen automates a lot of things in sff loading / creation. This fact makes Nomen not too hard to use, even if it manages in an unique way both sffv1 and sffv2.

Nomen is the first Mugen editor that allows you to edit both sff 1.0 and sff 2.0 files.

While automation helps a lot on making a coherent environment and, in the same time, helps users because avoids to ask them to do a lot of actions for every sff editing operation.

On the other hand, the automation system used by Nomen require a careful user. Infact there are some differences from how Nomen works and how Elecbyte official tools works. For more details see “Loading, Creating and Editing Sff”
